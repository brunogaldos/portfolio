AI Chatbot with RAG for Documents

Intelligent document processing and conversation system using Retrieval-Augmented Generation

Overview

This project implements a Retrieval-Augmented Generation (RAG) chatbot with dynamic document upload. Users can upload their own .txt files and chat with an AI that answers questions based on the uploaded content. The system is designed for accurate, context-aware document Q&A and is built with a modern, real-time chat interface.

Key Features

  • Dynamic document upload with drag-and-drop interface
  • RAG model with Dense Passage Retrieval for accurate document search
  • Real-time, responsive chat UI
  • Multi-file management and status monitoring
  • Hybrid search (FAISS + BM25/TF-IDF) and reranking for best results
  • Works on desktop and mobile devices

Technical Architecture

  • Uploaded .txt files are chunked into smaller segments using semantic chunking
  • Hybrid search combines FAISS vector similarity with BM25/TF-IDF
  • BGE Reranker v2 M3 reranks retrieved documents for relevance
  • BART Large CNN generates contextual summaries based on reranked passages
  • Pipeline: Sentence Transformer → Hybrid Search → Reranker → Summarizer
