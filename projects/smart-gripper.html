Smart Gripper with Object Detection

Intelligent robotic gripper system with computer vision for adaptive object manipulation

Overview

This project demonstrates an adaptive, torque-controlled robotic gripper capable of handling a wide spectrum of objects—from fragile balloons to heavy metal rods—without manual reconfiguration. The system combines precision magnetic sensing, adaptive PID torque control, lightweight deep learning, and an intuitive GUI to achieve both hands-on versatility and vision-guided autonomy.

Key Features

  • Torque-controlled gripping with PID and safety clamp
  • Vision-based object classification using a YOLO-style CNN
  • Automatic and manual operation modes via a two-tab GUI
  • Real-time feedback and adaptive force control
  • Online fine-tuning for new object classes

Technical Details

  • Task 1: Sensor Filtering and Torque Control
    • Zero-bias correction and median filtering of Z-axis magnetic readings
    • PID controller (kp=2.5, ki=0.5, kd=0.1) for smooth, stable closing
    • Safety clamp enforces voltage limits and automatic STOP on contact detection
  • Task 2: Vision-Based Object Classification
    • YOLO-style CNN trained on ~100 images (metal, wood, balloon, background)
    • Hard objects trigger high-torque routine; soft objects use low-force PID sequence
    • Model can be incrementally retrained online
  • GUI Modes
    • Manual: Hard/Soft presets, Open/Close buttons, real-time status bar
    • Automatic: Live camera feed, auto-actuation, action log, pause control
