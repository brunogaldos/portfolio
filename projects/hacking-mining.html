REN Mining Process Prediction

Recurrent Equilibrium Network for advanced mining process optimization and prediction

Overview

This project was developed during the Hackmining event to optimize the crusher process for a mining corporation. It leverages a Recurrent Equilibrium Network (REN) to predict new bin levels by combining historical data with current control inputs, such as traffic light status and feeder motor signals. The model learns an equilibrium representation that captures the system's temporal patterns and uses Broyden’s method to solve for the hidden variable reflecting the dynamic state. Training is performed using Mean Squared Error (MSE) loss and the Adam optimizer.

Key Features

  • Recurrent Equilibrium Network (REN) for time-series prediction
  • Combines historical and real-time control signals
  • Captures temporal patterns in mining process data
  • Uses Broyden’s method for dynamic state estimation
  • Trained with MSE loss and Adam optimizer

Usage

  1. Place your dataset (e.g., merged.csv or export_1740063344367.parquet) in the project directory.
  2. Run the preprocessing and training script:
    python data_preprocessing_REN.py
  3. The script will display training progress and save model checkpoints (e.g., model_epoch_1.pth).
  4. To plot and compare model outputs and targets, run:
    python test.py
